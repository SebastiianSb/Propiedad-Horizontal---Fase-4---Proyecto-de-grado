# Propiedad Horizontal

**Nota Importante:** Este proyecto es un prototipo desarrollado especificamente para la fase 3 del proyecto de grado.

## Descripcion General
Aplicacion web para la gestion integral de propiedad horizontal (conjuntos residenciales o edificios). El sistema esta disenado para facilitar la interaccion y administracion a traves de diferentes modulos basados en roles.

## Roles del Sistema
- **Administrador:** Modulo para la gestion de la propiedad, residentes y configuraciones generales del sistema.
- **Residente:** Modulo para el acceso a la informacion personal, servicios de la unidad y comunicacion.
- **Guardia:** Modulo para el control de acceso, registro de visitantes y seguridad del recinto.

## Estructura del Proyecto
- `index.html`: Pagina principal y de inicio de sesion.
- `admin.html`: Interfaz del rol de administrador.
- `resident.html`: Interfaz del rol de residente.
- `guard.html`: Interfaz del rol de guardia de seguridad.
- `css/`: Hojas de estilo para el diseno de la aplicacion.
- `js/`: Archivos de logica de programacion e interactividad.
